#   Chennai Super Kings  (sibling IPL franchise — different home venue)
#   India                (men's international — multi-tournament arc)
#   Australia            (men's international — bilateral + ICC events)
#
# `as_of_date` is pinned on every URL so calendar form windows
# (last_60d / last_6mo / last_1yr) are deterministic across runs.
# Production callers omit it; the regression harness pins it
# because md5-diff requires byte-stability.
#
# `FilterParams.filter_team` is IGNORED on these endpoints — the
# team path-param dominates. So no `filter_team=` here; the team
# is always in the path. `filter_opponent`, `filter_venue`, and
# every other axis apply normally.
#
# All REG — documents current behaviour of the v1 endpoint.
