# Scope averages — URL inventory for the new /api/v1/scope/averages/*
# router family that powers the "average team" column on the Teams >
# Compare tab. See internal_docs/spec-team-compare-average.md.
#
# All scope-averages URLs are NEW — they don't exist in HEAD, so
# byte-identical between HEAD and patched is suspicious (means the
# new router didn't actually register).
#
# A few REG entries hit existing team endpoints to confirm that the
# `_team_innings_clause` and `_partnership_filter` refactor (those
# helpers now accept team=None) is behaviour-preserving when team is
# given — the team endpoints stay byte-identical.
#
# Filter combos — match what the new router needs to handle:
#   - unfiltered (whole DB)
#   - tournament-scoped (IPL)
#   - season-scoped (2024)
#   - mixed (tournament + season + venue)
#   - team_type / gender combos (women's club)
