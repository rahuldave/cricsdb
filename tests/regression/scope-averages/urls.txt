# Scope averages — URL inventory for the /api/v1/scope/averages/*
# router family that powers the "average team" column on the Teams >
# Compare tab. See internal_docs/spec-avg-column-per-innings.md.
#
# 2026-04-26 — bulk REG → NEW flip for the per-innings transform
# (spec-avg-column-per-innings.md, Commit 1). Every endpoint that
# returns a numeric value affected by the per-innings transform OR
# carries chip envelopes whose `scope_avg` baseline changes is now
# NEW. Match-level results endpoints (`/scope/averages/summary`,
# `/teams/{team}/summary`) and team-side flat-by-season endpoints
# stay REG — they're untouched by the transform.
#
# Filter combos — match what the new router needs to handle:
#   - unfiltered (whole DB)
#   - tournament-scoped (IPL)
#   - season-scoped (2024)
#   - mixed (tournament + season + venue)
#   - team_type / gender combos (women's club)
